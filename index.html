Magic Row In Action (Fully Responsive)

The magic not only changes the number of rows for each breakpoint, but changed the width and padding of the last container in each row so that the content of each row item appears to be the same width

I set a max-height on these items so you can easily see what the grid is doing. Here is the Scss for the grid:

.magic {
	@include pinkmagic( $rowitems: 6, $after: 5 );

	@for $i from 7 through 8 {

		&:nth-of-type( 11n+#{$i} ) {
			@include pinkmagic( $rowitems: 2, $before: 6, $after: 3 );
			background-color:#bada55;
		} 
	}

	@for $i from 9 through 11 {

		&:nth-of-type( 11n+#{$i} ) {
			@include pinkmagic( $rowitems: 3, $before: 8 );
			background-color:blue;
		} 
	}
}
